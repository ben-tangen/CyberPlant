# CyberPlant TODO Dependency Board

This folder tracks who owns each task, what blocks it, and what work it unblocks.

## How to use
- `Status`: `Not Started`, `In Progress`, `Blocked`, or `Done`
- `Blocked By`: Task IDs that must be completed first
- `Unblocks`: Task IDs that can start after this task is done

## Files
- `TEAM_TASKS.md`: Full cross-team dependency map
- `PEYTON.md`: Peyton's task queue and dependencies
- `BEN.md`: Ben's task queue and dependencies
- `JUDAH.md`: Judah's task queue and dependencies
- `DONE_LOG.md`: Completed task history and what each completion unblocked
