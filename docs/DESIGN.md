# Design Notes

## Combat Feel
- Fast, responsive
- Skill-based (timing + dodging)
- Enemies use patterns

## Weapons
- Plant-based
- Each weapon has a unique use case
- Encourage switching

## Difficulty Philosophy
- Challenging but fair
- Learnable enemy patterns
- Bosses require adaptation

## World
- Cyberpunk + nature contrast
- Bright visuals with underlying decay

## Open Questions
- Should enemies drop health?
- How often should enemies spawn?
- How strong should upgrades feel?
