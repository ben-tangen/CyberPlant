# Final Game Project

**Team:** Ben, Judah, Peyton

## Game Overview
2D side-scrolling action game inspired by *Hollow Knight*, focused on combat, movement, and boss fights.

- Pixel-art, cyberpunk world with environmental decay
- Water = currency
- Plant-based weapons = progression

## Gameplay Loop
1. Explore level
2. Fight enemies
3. Collect water
4. Upgrade abilities
5. Defeat boss

~1 hour playthrough with replayability
